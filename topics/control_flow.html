Control Flow

Control the path your code takes. Learn conditionals with if/elif/else, and master iteration with for and while loops — all with animated visualizations.

Conditionals — if / elif / else

Conditionals let your program make decisions. Python uses indentation (4 spaces) instead of curly braces to define code blocks.

Python 
age = 18

if age < 13:
    print('Child')
elif age < 18:
    print('Teenager')
else:
    print('Adult')  # Output: Adult

Loops — for & while

Loops let you repeat code. A for loop iterates over a sequence; a while loop repeats while a condition is true.

Python 
# for loop over a list
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(fruit)

# range() generates a sequence of numbers
for i in range(5):
    print(i, end=' ')  # 0 1 2 3 4

# while loop
count = 0
while count < 3:
    print(f'Count: {count}')
    count += 1
