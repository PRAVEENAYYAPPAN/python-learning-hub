# Python Learning Hub

An interactive web platform for learning Python with animations, games, and hands-on coding exercises. Built entirely with free and open-source technologies.

## Features

- Interactive lessons with animated visualizations
- Live Python code editor (using CodeMirror)
- Mini-games and quizzes for reinforcement
- Visual explanations of programming concepts
- Responsive design for all devices

## Topics Covered

1. Variables and Data Types
2. Control Flow (Conditionals and Loops)
3. Functions
4. Data Structures (Lists, Dictionaries, Sets, Tuples)
5. Object-Oriented Programming (Classes, Inheritance, Polymorphism)

## Technologies Used

- HTML5, CSS3, JavaScript
- [p5.js](https://p5js.org/) - For animations and visualizations
- [CodeMirror](https://codemirror.net/) - For code editing
- [marked.js](https://marked.js.org/) - For markdown rendering
- All technologies are free and open-source

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Navigate through the topics and start learning!
