Variables & Data Types

Variables are the building blocks of every program. In this module you'll learn how Python stores data, what types are available, and how to work with them using f-strings, type checking, and dynamic typing.

⏱ ~20 minutes 📝 4 sections 🎯 1 quiz 💻 1 live editor

What are Variables?

A variable is a named container that holds a value. In Python, you don't need to declare a type — Python figures it out automatically (this is called dynamic typing).

Think of a variable as a labeled box: you put something inside, and you use the label to find it later.

Python 
# Assigning variables
name       = 'Alice'
age        = 25
height     = 1.75
is_student = True

print(name, age, height, is_student)
# Output: Alice 25 1.75 True

Core Data Types

Python has several built-in data types. The most common ones are:

🔢 int
Whole numbers: 42, -7, 0
🔟 float
Decimal numbers: 3.14, -0.5
📝 str
Text: "Hello", 'Python'
✅ bool
Logical: True, False
Python 
# Checking types with type()
x = 42
y = 3.14
z = 'Python'
b = True

print(type(x))  # <class 'int'>
print(type(y))  # <class 'float'>
print(type(z))  # <class 'str'>
print(type(b))  # <class 'bool'>

f-Strings & String Formatting

f-strings (formatted string literals) are the modern way to embed expressions inside strings. Prefix a string with f and use {} to insert variables.

Python 
name = 'Alice'
age  = 25

# f-string (Python 3.6+)
print(f'My name is {name} and I am {age} years old.')
# My name is Alice and I am 25 years old.

print(f'Next year I will be {age + 1}.')
# Next year I will be 26.
