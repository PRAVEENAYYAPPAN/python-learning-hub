## ✨ Features

| Feature | Description |
|---|---|
| 🌙 **Premium Dark UI** | Glassmorphism design with animated particle background |
| 💻 **Live Code Editor** | CodeMirror-powered Python editor in every module |
| 🎯 **Interactive Quizzes** | 3-question quiz per module with instant feedback |
| 📊 **Progress Tracking** | localStorage-based progress indicator on the home page |
| 📱 **Fully Responsive** | Works beautifully on desktop, tablet, and mobile |
| ⚡ **Zero Dependencies** | No build tools, no npm — just open index.html |
| 🔤 **Syntax Highlighting** | Custom hand-crafted syntax highlighting on all code blocks |
| 🎨 **Animated Flowcharts** | Canvas-based visualizations (Control Flow module) |

## 🚀 Getting Started

No setup required — simply open the project:

```bash
# Clone the repository
git clone https://github.com/PRAVEENAYYAPPAN/python-learning-hub.git
cd python-learning-hub

# Open in browser (any of these)
start index.html          # Windows
open index.html           # macOS
xdg-open index.html       # Linux
```

Or use **VS Code Live Server** for the best experience.

## 🏗️ Project Structure

```
python-learning-hub/
├── index.html                  # Homepage with hero, course grid & progress
├── css/
│   └── style.css               # Full design system (variables, components, layout)
├── js/
│   ├── main.js                 # Navbar, scroll-reveal, progress tracker, counters
│   └── particles.js            # Particle animation canvas engine
└── topics/
    ├── variables.html          # Module 01 — Variables & Data Types
    ├── control_flow.html       # Module 02 — Control Flow + animated flowchart
    ├── functions.html          # Module 03 — Functions
    ├── data_structures.html    # Module 04 — Data Structures + comparison table
    ├── oop.html                # Module 05 — OOP
    └── file_handling.html      # Module 06 — File Handling + JSON
```
