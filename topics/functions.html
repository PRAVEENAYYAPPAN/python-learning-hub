Functions

Functions are the backbone of reusable code. Learn to define functions, pass arguments, use default parameters, unpack *args and **kwargs, and write elegant lambda expressions.

⏱ ~30 minutes 📝 4 sections 🎯 1 quiz

Defining & Calling Functions

A function is a reusable block of code. Use def to define it and call it by name with ().

Python 
def greet(name, greeting='Hello'):
    return f'{greeting}, {name}!'

print(greet('Alice'))          # Hello, Alice!
print(greet('Bob', 'Hi'))   # Hi, Bob!

*args and **kwargs

*args collects extra positional arguments into a tuple; **kwargs collects keyword arguments into a dictionary.

Python 
def total(*nums):
    return sum(nums)

def introduce(**info):
    for k, v in info.items():
        print(f'{k}: {v}')

print(total(1, 2, 3, 4))   # 10
introduce(name='Alice', age=25)

Lambda Expressions

A lambda is a small anonymous function written in a single line — great for short operations.

Python 
# Lambda — anonymous one-line functions
square = lambda x: x ** 2
add    = lambda a, b: a + b

print(square(5))    # 25
print(add(3, 7))    # 10

# Sort by string length
names = ['Charlie', 'Alice', 'Bob']
names.sort(key=lambda n: len(n))
print(names)  # ['Bob', 'Alice', 'Charlie']
