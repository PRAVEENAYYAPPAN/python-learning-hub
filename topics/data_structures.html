Data Structures

Learn about Python's built-in data structures: lists, dictionaries, sets, and tuples.

Lists

Lists are ordered, mutable collections of items.

Try it Yourself

Dictionaries

Dictionaries store data as key-value pairs.

Sets and Tuples

Sets are unordered collections of unique items. Tuples are ordered, immutable collections.

Set Example:

# Set example
numbers = {1, 2, 3, 2, 1}
print(numbers)  # Output: {1, 2, 3}
numbers.add(4)
print(numbers)  # Output: {1, 2, 3, 4}

Tuple Example:

# Tuple example
point = (10, 20)
print(point[0])  # Output: 10
# point[0] = 15  # This would cause an error

Quick Quiz

What is the output of the following code?
my_list = [1, 2, 3, 4, 5]
print(my_list[1:4])
