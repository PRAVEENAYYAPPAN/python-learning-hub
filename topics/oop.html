Classes & Objects

A class is a blueprint. An object is an instance created from that blueprint. The __init__ method runs when an object is created.

Python 
class Dog:
    def __init__(self, name, breed):
        self.name  = name
        self.breed = breed

    def bark(self):
        return f'{self.name} says: Woof!'

    def __repr__(self):
        return f'Dog({self.name}, {self.breed})'

dog = Dog('Rex', 'Lab')
print(dog.bark())  # Rex says: Woof!
print(dog)         # Dog(Rex, Lab)

Inheritance

Inheritance allows a child class to reuse code from a parent class and extend or override its behaviour.

Python 
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        raise NotImplementedError

class Cat(Animal):
    def speak(self):
        return f'{self.name} says: Meow!'

class Duck(Animal):
    def speak(self):
        return f'{self.name} says: Quack!'

animals = [Cat('Whiskers'), Duck('Donald')]
for a in animals:
    print(a.speak())

Encapsulation & Properties

Use @property to create smart getters/setters that validate data before storing it.

Python 
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner    = owner
        self._balance = balance

    @property
    def balance(self):
        return self._balance

    def deposit(self, amount):
        self._balance += amount
        print(f'Balance: ${self._balance}')
